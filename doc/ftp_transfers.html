Quick guide: how to use ProHits to transfer raw files via FTP

  1. From the storage tab of the Data Management module of ProHits, go to the desired folder, Select the download arrow.
  2. Selecting the download arrow prompt a new window. Here, select "add the file in exporting list". Multiple files can be added to the export list.
  3. Once this is done, go back to the main ProHits window. At the bottom of the left side menu is a link to the "Export Raw Files" function.


  4. Clicking "Export Raw Files" brings up a new window. To transfer selected export files to remote computers, select "FTP server".
  5. This will expand the window in which you can input your FTP credentials. Once done, press "test connection".
  6. If the connection is working, the following options will appear... just press: "upload files". Tada!
